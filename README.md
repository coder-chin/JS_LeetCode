# JS_LeetCode

> 为什么创建本仓库？

> 原来一直用的 C++，但是在面试的过程中发现面试官更倾向于让应聘者使用 JavaScript。
> 不仅考察算法，也考察了应聘者的 JS 功底 (而且作为一名前端，有些面试官很可能是不懂 C++的).
